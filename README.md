## Project Overview

This project analyses the Linux Kernel as an example of a large-scale open source software system.  
The repository contains a set of Bash scripts designed to demonstrate:

- System identity and kernel information
- Open source package inspection
- Disk and permission auditing
- Log file analysis
- A generated open source manifesto

These scripts illustrate how open source tools and Linux utilities can be combined to perform system analysis and auditing tasks.
