## Project Overview

This project analyses the **Linux Kernel**, one of the most influential open source software projects in the world.

The repository demonstrates how open source tools and Linux utilities can be used to audit and analyse system behaviour through Bash scripting.

The project contains several scripts that examine system identity, inspect packages, analyse log files, and generate an open source manifesto.

## Scripts Description

### Script 1 – System Identity Report

Displays system information including:

* OS distribution
* Kernel version
* Logged-in user
* System uptime
* Licence information of the Linux kernel

---

### Script 2 – FOSS Package Inspector

Checks whether a package is installed and displays:

* Package metadata
* Package version
* Maintainer information
* Open source philosophy note related to the package

---

### Script 3 – Disk and Permission Auditor

Audits key Linux directories and reports:

* Directory permissions
* Owner and group
* Disk usage
* Kernel module directory details

---

### Script 4 – Log File Analyzer

Reads a log file and:

* Counts occurrences of a keyword
* Displays matching log entries
* Provides a summary of findings

---

### Script 5 – Open Source Manifesto Generator

Prompts the user for input and generates a personalised **Open Source Manifesto** saved as a text file.
